rLevelDB

Description

      Ruby extension for LevelDB
      http://code.google.com/p/leveldb/

Install

      $ git clone git://github.com/kajiken/rleveldb.git
      $ cd rleveldb/rleveldb
      $ ruby extconf.rb

Usage

sample ruby code. 
      require './../leveldb'
      require 'pp'

      db = LevelDB::DB.new

      db.open "/tmp/testdb"

      db.put "tokyo", "hogehgoe"

      pp db.get "tokyo"

      db.del "tokyo"

      pp db.get "tokyo"

      db.close
